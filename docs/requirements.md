# 功能需求文档

## 项目名称

多伦多大学选课规划助手（Course Planner for UofT）

## 目标用户

不懂代码的小白学生，需要在手机和电脑上都能使用的课程规划工具。

---

## 核心功能

### F1 — 学生信息输入

| 项目 | 说明 |
|------|------|
| 学年选择 | 大一 / 大二 / 大三 / 大四 |
| 专业组合 | 支持 1 Specialist / 2 Majors / 1 Major + 2 Minors |
| Focus 选择 | 部分专业支持 Focus（方向细分），可选 |
| 实时更新 | 修改任何参数后自动刷新规划，无需点击提交 |

### F2 — 课程推荐与规划

- 根据专业组合，自动列出：
  - **大一进专业前置课程**（用户为大一新生时）
  - **大二及以上必修课程**（按推荐学年分组）
  - **Focus 差异化课程**（不同 Focus 的不同路径）
- 每门课程展示：
  - 课程代码（如 CSC110H1）
  - 课程名称
  - 简短内容介绍（1-2 句话）
  - FCE 值（H=0.5, Y=1.0）

### F3 — 学分规则

| 规则 | 说明 |
|------|------|
| 毕业总学分 | 20 FCE |
| Overlap 检测 | 两个 Program 组合时，至少 12 FCE 必须唯一。最大重叠 = (A总FCE + B总FCE) − 12。超出则红色警告 |
| Breadth 广度 | 5 个类别，两种达标标准可选（方案 A / 方案 B） |

### F4 — 学期负载

- 每学期最多 **6 门课**
- 推荐 **5 门/学期**
- 自动按推荐值排课，超出时高亮警告标签
- 每学年包含：Fall 学期 + Winter 学期 + Summer 学期（默认空）

### F5 — 对比功能（核心特色）

- 保存 2~3 个不同方案
- 页面并排显示，对比维度：
  - 总课程列表
  - 每学期课程分布
  - 学分统计（已完成 / FCE 剩余）
  - Overlap 警告
  - Breadth 满足进度
- 支持添加/删除方案
- 每个方案独立配置

### F6 — 导出（后续版本）

- 导出对比方案为图片或 PDF
- 第一版暂不实现

---

## 用户交互

| 交互 | 行为 |
|------|------|
| 修改学年下拉框 | 实时更新所有方案 |
| 修改专业选择 | 实时更新对应方案 |
| 修改 Focus 选择 | 实时更新课程列表 |
| 点击课程卡片 | 展开/收起课程详细介绍 |
| 添加方案按钮 | 新建一个方案卡片（最多 3 个） |
| 删除方案按钮 | 移除方案卡片（最少 1 个） |
| 切换 Breadth 标准 | 实时更新广度进度条 |

---

## 非功能需求

- 零外部依赖（Google Fonts CDN 可选）
- 自包含单文件：复制 `.html` 即可使用
- 桌面端和移动端自适应
- 数据以 JS 对象硬编码，结构清晰便于替换
- 初始加载时间 < 1 秒
